# Checkers in Elm

## Development Instructions
1. Install VirtualBox
1. Install `vagrant`
1. Install `ansible`
1. Go to project folder
1. `vagrant up`
1. `vagrant ssh`
1. `cd /vagrant`
1. `make start`
1. Go to [localhost:3000](http://localhost:3000/)
